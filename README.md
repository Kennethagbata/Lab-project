# Lab-project
IQ App provides an accessible, user-friendly quiz platform that eliminates difficulty in creating, sharing and taking quizzes with multimedia support. 

Features
•	The Quiz App comes packed with features to enhance the user experience:
•	User Authentication
•	Sign-Up: Email, password, and username with password strength feedback.
•	Sign-In: Secure login with JWT authentication.
•	Forgot Password: Password reset functionality.
Quiz Management
•	Create Quiz: Add titles, descriptions, questions, multimedia (images/videos), timers, and points.
•	Edit Quiz: Save drafts and update quizzes anytime.
•	Quiz Taking
•	Start Quiz: Accessible start button with instructions.
•	Question Navigation: Progress indicators and clear answer tracking.
•	Timer: Countdown with pause/resume functionality.
•	Points System: Real-time scoring and feedback.
Feedback and Results
•	Scoreboard: Breakdown of correct/incorrect answers
•	Share Results: Share achievements on social media.
Additional Features
•	Leaderboard: Dynamic rankings based on scores and time.
•	Responsive Design: Works seamlessly on mobile, tablet, and desktop
